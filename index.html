Mathematical Melody Maker

Mathemelody is a musical sequencer that transforms mathematical equations into melodies. Each grid cell represents a step in the sequence, where the equation's result determines the note played.

How to Use
  1. Set the number of grids and click "Update Grid".
  2. Enter equations in each cell (use 'x' for the step number).
  3. Select a wave type from the dropdown menu.
  4. Click "Play" to start the sequence.
  5. Adjust tempo with the slider.
  6. Click "Stop" to end playback.

Example equation: 2*x + 1
